Typical Workflow (from project docs)
====================================

Open the following terminals from repo root:

1) DockerCompose
----------------
- First time (or when cert changed):
  - `pwsh -File ./docker/create-dev-cert.ps1`
- Start full stack (app + postgres + redis + seq):
  - `docker compose up --build`

Useful URLs after startup:
- App (HTTPS): `https://localhost:7186`
- Seq UI: `http://localhost:8081`
- Redis Insight UI: `http://localhost:5540`

2) Migrations
-------------
Note: ensure DB port in config/env matches your setup (usually 5432 or 5433).

- Add migration:
  - `dotnet ef migrations add <MigrationName> --project BlazorAutoApp --startup-project BlazorAutoApp --output-dir Data\Migrations`
- Apply migrations:
  - `dotnet ef database update --project BlazorAutoApp --startup-project BlazorAutoApp`

3) Tailwind
-----------
Run from `BlazorAutoApp.Client`:

- First time only:
  - `npm install tailwindcss @tailwindcss/cli`
- Watch/build CSS:
  - `npx @tailwindcss/cli -i .\Styles\input.css -o ..\BlazorAutoApp\wwwroot\tailwind.css --watch`
